# Windows Version

The Windows PowerShell version of mklinks is coming soon.

## Planned Windows implementation:
- PowerShell script with same functionality
- Windows path handling (backslashes, drive letters)
- NTFS symlink support
- WSL compatibility

## Contribution Welcome!

If you'd like to contribute a Windows version, please:
1. Follow the same configuration file format
2. Implement the same core functionality
3. Test on Windows 10/11
4. Submit a pull request
