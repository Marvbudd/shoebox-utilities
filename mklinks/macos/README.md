# macOS Version

The macOS version of mklinks is coming soon. For now, you can use the Linux version which should work on macOS:

```bash
cd /path/to/your/media
../linux/mklinks mklinks.conf
```

The Linux version uses standard bash commands that are compatible with macOS.

## Planned macOS-specific features:
- Finder integration
- macOS path handling
- Notification Center alerts on completion
